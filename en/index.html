Domain Name Word Chain

Domain name word chain

Rules

  • Play word chains with domain names.

    Examples: www.google.com, youtube.com, wikipedia.org

  • English letters (a-z), numbers (0-9), hyphens (-) and dots (.) can be used.

  • If the domain doesn’t exist, you lose a life.?
    Check Public DNS and assume that a domain with a DNS response code of 0 exists.

  • Cannot use the same domain.

    *Note: www.google.com and google.com are considered different.

  • The game ends after losing three lives.

  • The game starts with www.google.com, the world’s most famous domain.

  • Points are based on the domain's length and other factors.

  • No searching allowed!
